# 📋 Шалгалт Системийн Хурдан Гарын Авлага

## Үндсэн Асуулт: "Quiz ID гэж юу вэ?"

### 🔨 ӨМНӨ vs ОДОО

| Аспект | ӨМНӨ | ОДОО |
|--------|------|------|
| **Хадгалалт** | `quizKey: "quiz_g6"` (текст) | `quizId` → Quiz ID (объект) |
| **Асуулт** | ❌ Асуулт, хариуц хранилуу дарав | ✅ Quiz ба QuizQuestion модель |
| **Оноо** | ❌ Оноо заас боломжгүй | ✅ Бүр асуултанд оноо |
| **Төрөл** | ❌ үүгүй | ✅ assignment/homework/exam |
| **Даалгавар** | ❌ хүүхдэд хүүхэл дүчүүлэн огт | ✅ Гэрийн даалгавар байдаг |

## 🎯 Хэрэглэгчийн Зүгээс

### Admin Panel (http://localhost:3000/admin)

Хоёр Tab байдаг:

#### 📖 **Хичээлүүд Tab**
- Хичээл үүсгэх/засах/устгах
- HTML агуулга оруулах
- **🎯 Шалгалт сонгох** ← Шалгалт холбоно

#### ✏️ **Шалгалт & Даалгавар Tab**  
- Шалгалт үүсгэх/засах/устгах
- Төрөл сонгох: assignment/homework/exam
- Асуулт нэмэх (future feature)

## 📚 Ашиглалтын Урсгал

```
STEP 1: Шалгалт үүсгэх
┌────────────────────────────────┐
│ ✏️ Шалгалт Tab дарна           │
│ → ✨ Шинэ шалгалт дарна       │
│ → Шалгалтын нэр оруулна       │
│   ├─ "6-р ангийн үсэг"         │
│   ├─ Анги: 6                   │
│   ├─ Төрөл: assignment         │
│   └─ Оноо: 100                 │
│ → ✓ Үүсгэх                     │
└────────────────────────────────┘

STEP 2: Хичээл үүсгэх + Шалгалт холбоно
┌────────────────────────────────┐
│ 📖 Хичээлүүд Tab дарна         │
│ → ✨ Шинэ хичээл дарна         │
│ → Хичээлийн нэр оруулна        │
│   ├─ "Үсэг таних үндэс"        │
│   ├─ Анги: 6                   │
│   ├─ Төрөл: lesson             │
│   ├─ HTML Агуулга орулна       │
│   └─ 🎯 Шалгалт сонгоно       │
│      [Dropdown] →              │
│      "6-р ангийн үсэг"         │
│ → ✓ Үүсгэх                     │
└────────────────────────────────┘

RESULT: Хичээл → Шалгалт → Асуултууд холбогдсон
```

## 🔗 Холбоис Төлөвлөө

```
Хичээл {
  id: "lesson_123"
  title: "Үсэг таних үндэс"
  quizId: "quiz_456"  ← ШИНЭ (өмнө: quizKey)
}

Quiz {
  id: "quiz_456"
  title: "6-р ангийн үсэг таних"
  grade: 6
  type: "assignment"
  totalPoints: 100
}

QuizQuestion[] {
  {
    quizId: "quiz_456"
    question: "Ямар үсэг вэ?"
    points: 10
  },
  {
    quizId: "quiz_456"
    question: "Хэрхэн уншдаг вэ?"
    points: 20
  }
}
```

### Сонгох Цөцгөлөл

- `quizId` (ID) → Database дээр Quiz объект
- `quizKey` (текст) → энгийн текст (УСТГАЛАА)

## 🎯 Гол Өөрчлөлтүүд

✅ **Үүсгэгдсэн**

- Lesson ↔ Quiz холбоис

❌ **Баасан**
- `quizKey` (энгийн текст) → quizId (объект холбоис)
