# 📚 Шалгалт ба Даалгаврын Систем

## Асуудлаа Ойлгонд

**"Quiz ID гэж юу вэ?"** гэж асуусан байна. Энэ системд **"Quiz ID"** нь уг асуулт, хариуцын хэсэг юм.

### Өмнөх System (Quiz ID - Энгийн текст)
```
Хичээл → quizKey: "quiz_g6_lesson1" (энгийн текст)
```

#### Асуудал:
- ❌ Асуулт хариуцыг хранилуулах боломжгүй
- ❌ Оноог тодруулж болохгүй
- ❌ Даалгавар ба шалгалтын ялгаа困難

---

## 🎯 Шинэ System (Профессиональ Шалгалт)

### Структур

```
Шалгалт (Quiz)
├── Нэр: "6-р ангийн үсэг таних шалгалт"
├── Төрөл: assignment | homework | exam
├── Нийт оноо: 100
└── Асуултууд (Questions)
    ├── Асуулт 1: "Ямар үсэг вэ?" [10 оноо]
    ├── Асуулт 2: "Хэрхэн уншдаг вэ?" [10 оноо]
    └── Асуулт 3: "Үүнийг хэн ашигладаг?" [80 оноо]
```

---

## 📖 Хэрхэн Ашиглах

### 1️⃣ Администратор Панель Орох
```
http://localhost:3000/admin
```

### 2️⃣ Шалгалт Үүсгэх (✏️ Шалгалт & Даалгавар Tab)

**A. Шинэ шалгалт нээх**
- "✨ Шинэ шалгалт" товчийг дарна
- Цүнхээ бөглөнө:
  - **🎯 Нэр**: "6-р ангийн үсэг таних шалгалт"
  - **📊 Анги**: 6
  - **📝 Төрөл**: 
    - `assignment` = Байнгын даалгавар
    - `homework` = Гэрийн даалгавар
    - `exam` = Шалгалт
  - **💬 Тайлбар**: Энэ шалгалтын зорилго, агуулгын тайлбар
  - **📊 Нийт оноо**: 100 (эргүүлэх)

**B. Асуулт нэмэх**
- Шалгалт засахаас: Edit товчийг дарна
- Modal нээгдсэн дараа: "❓ Асуултууд" хэсэг дээр "+ Асуулт нэмэх" товч байдаг

---

### 3️⃣ Хичээлтэй Шалгалт Холбох

**A. Хичээл үүсгэх/засах (📖 Хичээлүүд Tab)**
- "✨ Шинэ хичээл" товчийг дарна
- Бөглөнө:
  - Хичээлийн үндсэн мэдээлэл (нэр, анги, төрөл)
  - **HTML Агуулга**: Хичээлийн материал (өнгөт текст, зургаас гэх мэт)
  - **🎯 Шалгалт сонгох**: Өмнө үүсгэсэн шалгалтыг сонгодог

### Жишээ Flow:
```
1. Шалгалт үүсгэх: "6-р ангийн үсэг таних шалгалт"
   ↓
2. Хичээл нэмэх: "Үсэг таних үндэс"
   ↓
3. Хичээлд шалгалт холбох: [сонгох] → "6-р ангийн үсэг таних шалгалт"
```

## 📊 Шалгалтын Төрүүлүүд

| Төрөл | Зорилго | Жишээ |
|------|---------|------|
| **assignment** | Байнгын жанжиглал | Үйлийн дасгал |
| **homework** | Гэрийн даалгавар | 5 номын үндшинж буцалгах |
| **exam** | Эцсийн тест | Сард 1 удаа |

## 💾 Database Structure

```sql
-- Quiz Table (Шалгалтуудын хүснэгт)
Quiz {
  id: String           -- Өвөрлөг ID
  title: String        -- "6-р ангийн үсэг таних шалгалт"
  description: String  -- "Үсгийн суурь элементүүдийг ойлгох"
  grade: Int          -- 6
  type: String        -- "assignment" | "homework" | "exam"
  totalPoints: Int    -- 100
}

-- QuizQuestion Table (Асуултуудын хүснэгт)
QuizQuestion {
  id: String          -- Өвөрлөг ID
  quizId: String      -- Аль шалгалтад хамаарах
  question: String    -- "Ямар үсэг вэ?"
  type: String        -- "multiple" | "short" | "essay"
  options: String[]   -- ["A", "B", "C", "D"]
  answer: String      -- "A"
  points: Int         -- 10
}

-- Lesson Table (Хичээлүүдийн хүснэгт) - UPDATED
Lesson {
  id: String
  title: String
  quizId: String    -- ӨМНӨ: quizKey → ОДОО: quizId (Шалгалтын ID)
}
```

## 🔌 API Endpoints

### Шалгалт нэрүүлэх
```
GET  /api/admin/quizzes?grade=6
POST /api/admin/quizzes              -- Шинэ шалгалт
PUT  /api/admin/quizzes/:id         -- Засах
DELETE /api/admin/quizzes/:id       -- Устгах
```

### Асуултуудыг нэрүүлэх
```
GET  /api/admin/quizzes/:id/questions
POST /api/admin/quizzes/:id/questions           -- Асуулт нэмэх
PUT  /api/admin/quizzes/:id/questions/:qId     -- Засах
DELETE /api/admin/quizzes/:id/questions/:qId   -- Устгах
```

### Хичээлүүдэд Шалгалт Холбоно
```
GET  /api/admin/lessons?grade=6
POST /api/admin/lessons
  {
    "grade": 6,
    "kind": "lesson",
    "title": "Үсэг таних үндэс",
    "quizId": "quiz_id_here"  -- ӨМНӨ: "quizKey" → ОДОО: "quizId"
  }
```

## ❓ Түгээмэл Асуултууд

### Q: Quiz ID нь хүний үг дээр юуг хэлдэг вэ?
**A:** Квиз гэрхэлэх ID юм. Өмнө нь энгийн текст ("quiz_g6_") байсан. Одоо Даалгавр->Quiz модель байгаа тултай, асуулт, хариуцыг зохион байгуулж болно.

### Q: Шалгалтын төрлүүл ялгаа юу вэ?
**A:**
- `assignment` = Байнгын жанжуулах дасгал (класстаа хийнэ)
- `homework` = Гэрийн даалгавар (гэрээ хийнэ)
- `exam` = Эцсийн шалгалт (аль сард 1 удаа)

### Q: Асуулт яаж нэмэх?
**A:** Шалгалтыг засахаас (Edit) modal нээгдэх үед "❓ Асуултууд" хэсэг нээж, "+ Асуулт нэмэх" товч дарна.

### Q: Хичээлээс хичээл нь шалгалтыг хадгалахбүл хүн байна уу?
**A:** Үгүй. Admin-наас хичээлтэй шалгалт холбовол автоматаар хадгалагдана. Schema дээр: `Lesson.quizId` → `Quiz.id`
